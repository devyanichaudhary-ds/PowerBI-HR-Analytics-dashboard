# HR Analytics Dashboard (Power BI)

## Overview

This project analyzes employee data to help HR teams understand workforce demographics, employee attrition, and key factors affecting employee retention.

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Pages

### 1. Executive Overview
- Total Employees
- Active Employees
- Attrition Rate
- Average Age
- Average Income

### 2. Attrition Analysis
- Attrition by Department
- Attrition by Job Role
- Attrition by Salary
- Attrition by Overtime
- Attrition by Age Group

### 3. Workforce Demographics
- Gender Distribution
- Education Field
- Business Travel
- Department Distribution
- Job Role Distribution

### 4. HR Insights & Recommendations
- Business Travel Analysis
- Distance From Home Analysis
- Satisfaction Analysis
- HR Recommendations

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- DAX Measures
- Interactive Dashboards
- KPI Design
- Business Analysis
- Data Visualization

## Dataset

IBM HR Analytics Employee Attrition Dataset.
